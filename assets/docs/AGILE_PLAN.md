# Agile Sprint Plan - FitForge Gym

## Overview
Total Duration: 5 Sprints (approx. 2 weeks)
Goal: Deliver a fully responsive, static gym website.

| Sprint | Deliverable | Duration | Description |
| :--- | :--- | :--- | :--- |
| **Sprint-0** | **Discovery & Design** | Day 1 | Finalize SRS, Project Structure, Architecture, and UI Theme. |
| **Sprint-1** | **Core Framework** | Day 1-2 | Setup Project (HTML5/Tailwind), Header, Footer, Hero Section. |
| **Sprint-2** | **Features & Content** | Day 2-3 | Implement About, Membership Plans, and Trainers pages. |
| **Sprint-3** | **Visuals & polish** | Day 3-4 | Implement Gallery, Contact Page (with Map/WhatsApp), and Mobile Optimization. |
| **Sprint-4** | **Testing & Release** | Day 5 | Cross-browser testing, Light/Dark mode check (if applicable), and Deployment. |

## Detailed Breakdown

### Sprint-0: Planning (DONE)
- [x] Idea Validation
- [x] requirements Gathering (SRS)
- [x] Architecture Design

### Sprint-1: Core Structure
- **Objective**: Get the skeleton ready.
- **Tasks**:
    - Initialize Git Repo.
    - Setup Tailwind CSS (via CDN for simplicity or CLI).
    - Create `index.html`, `style.css`, `script.js`.
    - Build Responsive Navbar & Footer.
    - Build Hero Section with "Join Now" CTA.

### Sprint-2: Main Pages
- **Objective**: Add informational content.
- **Tasks**:
    - **About Us**: Mission, Vision, Why Choose Us.
    - **Plans**: Card layout for pricing.
    - **Trainers**: Profile cards.

### Sprint-3: Engagement & Assets
- **Objective**: Visual appeal and contact.
- **Tasks**:
    - **Gallery**: CSS Grid layout for images.
    - **Contact**: Form UI, Embedded Map, Floating WhatsApp Button.
    - **Assets**: Gather and optimize gym stock images.

### Sprint-4: Final Polish
- **Objective**: Zero bugs, fast load.
- **Tasks**:
    - Audit Lighthouse Score (Target: 90+).
    - Verify all links and buttons.
    - Final Code Cleanup.
